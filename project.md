## Это минималистичный вариант игры "3 в ряд" (aka match 3).

### Ключевые идеи:
 - простой дизайн и UI
 - только zen mode - без уровней, прогресса, энергии и т.п.
 - 6 видов фишек на лёгком уровне сложности, 7 - на среднем, 8 - на харде
 - визуальная тема фишек - крысиная еда
 - усилители:
    - "линейный" - даётся за 4 фишки в ряд, стирает всю строку или столбец
    - "бомба" - даётся за 5 фишек в форме T или L, стирает область 3x3
    - "цветной" - стирает все фишки выбранного типа
 - активация усилителей:
    - через swap
    - при стирании фишки-усилителя
 - комбо усилителей при swap:
    - усилитель + обычная фишка = стирание + срабатывание усилителя
    - усилитель + усилитель:
        - если два линейных, то стирание "крест-накрест"
        - иначе - последовательное срабатывание обоих усилителей

### План разработки:
**Этап 1:**
 - поле 8x8
 - заполнение поля случайными фишками
 - гарантия, что при начальном заполнении в поле не будет матчей (aka match)
 - но будут доступные ходы

**Этап 2:**
 - swap фишек
 - проверка матчей
    - если есть - найти подходящие фишки
    - удалить их
    - активировать их усиления (пока заглушкой)
    - досыпать новые в верхний ряд
    - снова проверить матчи (т.е. вызвать каскад/рекурсию)
    - если матча нет - откатить свапнутые фишки к исходному положению

**Этап 3:**
 - добавить список усилителей
 - в проверку матчей добавить поиск усилителей
 - если клетка определяется как усилитель - давать ей специальное свойство "power" с типо усиления
 - задействовать эти усиления
 - задействовать комбо

**Этап 4:**
 - сделать плавные анимации (короткие, по 150-200 ms)
 - пригладить дизайн
 - добавить поддержку тачскрина
 - завернуть приложение в PWA

### Стек:
 - JS
 - DOM
 - PWA
 - возможен React для красоты и удобства

### Гениальные идеи:
 - анимация при срабатывании усилителей: крыса делает НЯМ
